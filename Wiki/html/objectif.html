L'objectif du site

Dans le cadre de ce défi, vous êtes invités à automatiser une tâche redondante que l'un d'entre vous effectue régulièrement, afin d'optimiser votre temps. Nous avons opté pour la création d'un générateur de site web nommé GénéraSite. Ce projet offre la possibilité de personnaliser un site web selon vos préférences. En automatisant le processus de création de sites web, vous pourrez gagner du temps et bénéficier d'une flexibilité accrue dans la conception de vos projets en ligne.

Les contraintes

Votre projet devra respecter les contraintes suivantes :
- Être d'un niveau de difficulté convenable pour des B1 en informatique sur un second semestre

- Être pertinent vis à vis du thème "Cette fois, j'automatise!", c'est à dire créer un programme qui permette de faire quelque chose de rébarbatif habituellement chronophage en quelques secondes/minutes.

- Utiliser Git pour la gestion de versions et bien fusionner les branches de tout le monde si vous voulez que les contributions soient prises en compte.

- Disposer d'une interface utilisateur, soit en ligne de commande (prompts et attente de réponse, arguments de la ligne de commande et fichiers de configuration...) soit une interface web ou une autre interface graphique de votre choix

- Bénéficier d'une documentation sous la forme d'un Wiki, avec deux parties : un manuel d'utilisateur (comment utiliser le programme, ce qu'il fait), ainsi qu'une documentation technique (comment est organisé le code, comment lancer le projet)
